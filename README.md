SRMHive is a full-stack web platform that connects students and developers to collaborate on projects. Users can post project ideas, browse open opportunities, apply to join teams, and communicate in real-time through project-specific team chats.

Key Features:

🔐 JWT-based authentication (register, login, persistent sessions)
📋 Create & manage project posts (Hackathon, Startup, Academic, Personal)
🔍 Browse & filter projects by type, mode, and skill keywords
📩 Apply to join projects; owners can accept or reject applicants
💬 Real-time team chat per project using Socket.IO (restricted to accepted members)
🔔 Live in-app notifications for new applications and status updates
👤 Editable user profiles with skills, bio, GitHub & LinkedIn links
📊 Personal dashboard showing posts created, applications sent, and skills
Tech Stack:

Frontend: React, Vite, Tailwind CSS, React Router, Axios, Socket.IO Client
Backend: Node.js, Express.js, MongoDB (Mongoose), Socket.IO, JWT, bcrypt
